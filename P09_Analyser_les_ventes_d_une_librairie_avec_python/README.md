# 📚 Analyser les ventes d'une librairie avec Python

Bienvenue sur la page consacrée au projet **Analyser les ventes d'une librairie avec Python**. 

Ce projet illustre mes compétences en **Analyse de données, statistiques et Python** à travers l'analyse complète des ventes et des patterns d'achat d'une librairie.

Pour mener à bien ce projet j'ai :

* Exploré et visualisé les données de ventes
* Préparé et nettoyé les données commerciales
* Calculé les indicateurs clés de performance (KPIs)
* Analysé les corrélations entre variables
* Identifié les tendances et comportements clients
* Présenté les insights de manière professionnelle

---

## 📖 Contexte

Une librairie indépendante souhaite mieux comprendre sa clientèle et ses patterns de ventes pour optimiser son offre commerciale et ses stratégies marketing. Le projet vise à fournir une analyse approfondie basée sur des données réelles.

---

## 🎯 Objectif

Analyser les ventes de la librairie en :

- Décrivant les patterns de ventes
- Analysant le comportement d'achat des clients
- Identifiant les corrélations entre variables
- Calculant les indicateurs de performance clés
- Proposant des recommandations commerciales
